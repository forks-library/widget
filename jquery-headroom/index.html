$('.head').headroom({'hiddenTop':400,autoHide:false,'scrollOffset':60});

调用方法：

$(selector).headroom(options,callback);

options参数

参数 默认值 说明
hiddenTop [自动计算] 滚动到哪个位置开始隐藏菜单
fixedTop 0 浮动距离浏览器视窗顶部的位置
duration 500 隐藏展示动画的时间间隔
autoFixed true 是否自动浮动
autoHide true 是否自动隐藏
scrollOffset 0 描点滚动偏移顶部距离
zIndex 9999 图层z-index
isScrollActive true 滚动是改变选中状态
activeIndex 0 默认选中菜单
activeCls 'active' 菜单选中时的class
onResize function(api){} 窗口变化时触发

callback(api)参数

参数 默认值 说明
api.setOptions() - 修改默认参数
